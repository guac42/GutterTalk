# Gutter Talk

A bowling game app that insults you when you bowl poorly. Just like bowling with friends, GutterTalk provides feedback as you bowl. The goal is provide players with a unique and fun experience.

## Usage Instructions
- All of our buttons are self-explanatory. Right now, we just have our screens and navigation set up.
- Still working on the functionality. The login screen allows anyone to log in without writing anything.
